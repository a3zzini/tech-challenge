# Next Media - Tech challenge by Ayoub Ezzini

### Requirements :

> PHP >7.3
> NodeJS >12
> Composer

### Configuration files :

> .env (must be created if inexisted)


### Run those commandes :

```sh
> composer install
> npm install
> php artisan migrate
> php artisan serve
> npm run watch (optionel)
```
